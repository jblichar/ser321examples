You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply -- returns product of two values. Default values will be used for missing parameters.
    •    host:port/multiply?num1={number}&num2={number}

  • /github -- returns data of public repos for specified user
    •    host:port/github?query=users/{username}/repos

  • /volume -- returns the volume of a cone. Requires radius and height values.
    •    host:port/volume?rad={radius}&h={height}

  • /currency -- returns the latest currency conversion for provided currencies. Requires valid currency codes. Must be lowercase.
    •    host:port/currency?curr={choose currency to be converted}&conv{desired currency}

       example - host:port/currency?curr=eur&conv=jpy
